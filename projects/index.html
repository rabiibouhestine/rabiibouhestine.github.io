    What I do when I feel productive!

    Corona Virus Disease (COVID-19) Dashboard

    In this project, I visualised the worldwide situation of the Covid-19 epidemic.
    I used Google Sheets and aggregated data provided by Johns Hopkins University to automate sourcing. This data is aggregated and updated daily on this this github repository. Then used Tableau to make an interactive Dashboard.

    Go to the project page!

    World Happiness Report (2020) Dashboard

    In this project, I analysed the World Happiness Report data using Tableau to investigate the impact of 6 different factors on the happiness score. The happiness scores and rankings use data from the Gallup World Poll. The scores are based on answers to the main life evaluation question asked in the poll.

    Go to the project page!

    Twitter Analysis (NLP Web Application)

    In this project, I developed an R Shiny app that allows users to generate 2 NLP reports. The first report describes a Twitter account provided by the user, analysing the tweets made by that account. The second report analyses the tweets that contain a certain hashtag provided by the user.

    Go to the project page!
